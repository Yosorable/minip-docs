# previewImage

预览图片。

# 参数

| 名称 | 类型   | 必填 | 默认值 | 说明                             |
| ---- | ------ | ---- | ------ | -------------------------------- |
| url  | string | 是   |        | 图片 URL，目前版本只支持网络图片 |

# Example

```typescript
import { previewImage } from "minip-bridge";

previewImage("https://example.com/example.png");
```
