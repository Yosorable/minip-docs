# previewVideo

预览视频。

# 参数

| 名称 | 类型   | 必填 | 默认值 | 说明                             |
| ---- | ------ | ---- | ------ | -------------------------------- |
| url  | string | 是   |        | 视频 URL，目前版本只支持网络视频 |

# Example

```typescript
import { previewVideo } from "minip-bridge";

previewVideo("https://example.com/example.mp4");
```
