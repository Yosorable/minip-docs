# previewVideo

Preview Video.

# 参数

| Name | Type   | Required | Default | Description                                            |
| ---- | ------ | -------- | ------- | ------------------------------------------------------ |
| url  | string | Yes      |         | Video URL, currently only network images are supported |

# Example

```typescript
import { previewVideo } from "minip-bridge";

previewVideo("https://example.com/example.mp4");
```
