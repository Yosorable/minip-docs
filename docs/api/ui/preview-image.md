# previewImage

Preview Image.

# Parameters

| Name | Type   | Required | Default | Description                                            |
| ---- | ------ | -------- | ------- | ------------------------------------------------------ |
| url  | string | Yes      |         | Image URL, currently only network images are supported |

# Example

```typescript
import { previewImage } from "minip-bridge";

previewImage("https://example.com/example.png");
```
